# PrismaSLR Browser Extension
A lightweight Chrome Extension that helps researchers search academic papers using the Semantic Scholar API and automatically extract RIS files for integration into the PrismaSLR system.

## Features
- Search research papers via **Semantic Scholar API**
- Automatically extract **RIS citation files**
- Quick integration with PrismaSLR system
- Lightweight, built with **Vanilla JavaScript**
- Simple UI for fast academic paper lookup
  
## Tech Stack
- HTML5  
- CSS3  
- JavaScript  
- Chrome Extension Manifest V3  
- Semantic Scholar API  

## Installation (Developer Mode)

1. Clone the repository:
```bash
git clone repository-url
````

2. Open Chrome and go to:

```
chrome://extensions/
```

3. Enable **Developer Mode**

4. Click **Load unpacked**

5. Select the project folder

## How It Works
1. User searches for a research topic in the extension popup and filter papers by year range
2. Extension calls APIs from PrismaSLR Backend system included **Semantic Scholar API**
3. Results are displayed in a simple UI
4. User can export citation data as **RIS file**
5. RIS file can be imported into the PrismaSLR system
